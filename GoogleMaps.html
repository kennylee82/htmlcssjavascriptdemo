Google Maps API Example

Interactive map with markers, info windows, and controls

Note: This example requires a valid Google Maps API key. Replace "YOUR_API_KEY" in the script tag below with your actual API key from the Google Cloud Console.

Features Demonstrated:

  • Map initialization with custom center and zoom
  • Adding and removing markers
  • Info windows with custom content
  • Map type switching (roadmap/satellite)
  • Geocoding for address search
  • Event listeners for user interactions
